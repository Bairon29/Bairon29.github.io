Flexbox

Today, we will be learning about Flexible Box or flexbox - This is just an example

Flexible Box (flexbox) is design to achieve complex layout model that the traditional float and position property could not achieve easily

Flexbox consists of various properties that helps us define two types of boxes, flex containers and flex items. The flex containers have one or more flex items and defines how these flex items are position, order, and how they are wrapped. Flex items have the content

A div container can be made a flex container by providing the display property the value flex, when styling the container. Any container inside is consider a flex item display: flex;

Consider this example:

In this example we have one div container with three direct child. 

                    .flex_container
                    {
                    
                        background-color: black;
                        
                        width: 40%;
                        
                        height: 350px;
                        
                        margin: 10px auto;
                        
                        color: black;
                    
                    }
                    .flex_item
                    {
                        
                        background-color: beige;
                        
                        width: 100px;
                        
                        height: 100px;
                        
                        text-align: center;
                        
                        font-size: 60px;
                    }

Let's apply flex

To make the blaxk box a flex container, we just need to apply the display: flex;

                    .flex_container
                    {
                    
                        background-color: black;
                        
                        display: flex;
                        
                        width: 40%;
                        
                        height: 350px;
                        
                        margin: 10px auto;
                        
                        color: black;
                    
                    }
                    .flex_item
                    {
                        
                        background-color: beige;
